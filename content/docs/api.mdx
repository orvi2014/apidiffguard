A public `https://api.apidiffguard.com/v1` host and personal access tokens are **not available yet**.

## What exists today

| Surface | Notes |
| --- | --- |
| Console | Full baseline / check / diff / alerts / schedules UI |
| `POST /api/check` | One-off public URL probe (app origin) |
| `POST /api/openapi/parse` | OpenAPI parse for import |
| Stripe webhooks | Billing only |

API tokens under **Settings → Tokens** are listed as coming soon. Do not expect `adg_live_…` keys to work.

## Planned API (coming soon)

```
Authorization: Bearer adg_live_…
POST /v1/endpoints/:id/check
```

Until that ships, automate with the console, schedules (cron), and webhook alert channels.
