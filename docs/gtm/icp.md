# Ideal customer profile (ICP)

Don't sell to "developers." Sell to people who **depend on APIs every day**.

## Best-fit buyers

- SaaS founders integrating Stripe, Shopify, HubSpot, Twilio, etc.
- Agencies building API integrations for clients
- DevOps / platform engineers owning shared services
- Companies with multiple internal microservices
- Teams consuming partner / vendor APIs

## Signals they're in pain

- Public docs advertising "50+ integrations"
- Job posts mentioning API integrations or partner engineering
- Recent outages blamed on a third-party API
- Changelog complaints about unexpected response shapes

## Not a fit (yet)

- Solo hobby projects with a single static API
- Teams that only own APIs they fully control and never change
- Anyone looking for a full APM / tracing suite (we're contract/drift focused)
